# GrowthX SaaS Website 🚀

A modern and responsive SaaS website inspired by real-world platforms like HubSpot.

## 🔥 Project Overview

This project demonstrates a complete SaaS landing page including pricing plans, features, testimonials, and call-to-action sections.

## ✨ Features

* Responsive Design (Mobile Friendly)
* Pricing Plans UI
* Smooth Hover Animations
* Clean SaaS Layout

## 🛠️ Tech Stack

* HTML
* CSS
* WordPress (Elementor)
